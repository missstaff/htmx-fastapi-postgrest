# htmx-fastapi-postgrest

###### Learning how to build a fullstack applications with:
1. HTMX
2. Python
3. FastApi
4. Pydantic
5. Postgrest

### TO RUN THIS APPLICATION  🏃‍♀️
- `cd backend`
- **For macOS/Linux**:
    `source venv/bin/activate`
    
- **For Windows**:
    `.\venv\Scripts\activate`
    
- `pip install -r requirements.txt`
- `python run.py`
